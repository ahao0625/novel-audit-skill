# novel-audit-skill

网文审计技能 - 系统性审计网络小说文本，覆盖 11 个维度，附带具体改写建议。

## 功能特性

- **11个审计维度**：文字质量、AI痕迹、重复雷同、情节合理性、时间线一致性、上下文连贯性、叙事视角一致性、节奏与爽点、人物塑造、世界观自洽、对话质量
- **双模式**：单章详细审计 + 多章跨章对比审计
- **去AI化改写建议**：每条问题附带 ❌原文 → ✅建议 → 💡思路
- **加权评分**：A类×0.35 + B类×0.40 + C类×0.25，输出综合评级
- **题材覆盖**：玄幻/都市/悬疑/言情/历史等全题材通用

## 安装

将本仓库克隆到 WorkBuddy/Cursor 的 skills 目录即可：

```bash
# WorkBuddy
cp -r novel-audit ~/.workbuddy/skills/novel-audit

# 或手动将 novel-audit 文件夹放入技能目录
```

## 使用方法

在 AI 助手中输入以下触发词即可启动审计：

- `审计`、`审稿`、`检查这章`
- `帮我看看这段`、`有没有AI味`
- `前后矛盾吗`、`质量怎么样`

支持直接粘贴章节文本，或指定文件路径进行审计。

## 评分标准

| 综合分 | 评级 | 发布建议 |
|--------|------|----------|
| 8.5-10 | 优质 | 可直接发布，锦上添花式修改 |
| 7.0-8.4 | 良好 | 处理高优问题后可发布 |
| 5.5-6.9 | 及格 | 处理高优+中优问题后发布 |
| 4.0-5.4 | 待改进 | 需要较多修改，建议暂缓发布 |
| 0-3.9 | 需大改 | 建议重写或大幅修改 |

## 11个审计维度详解

| 类别 | 维度 | 说明 |
|------|------|------|
| **A类 基础质量** (×0.35) | A1 文字质量 | 错别字、语病、标点、流水账 |
| | A2 重复雷同 | 跨段落/跨章重复检测 |
| | A3 AI痕迹 | 词库/句式/结构三层检测 |
| **B类 叙事逻辑** (×0.40) | B1 情节合理性 | 逻辑链、动机、因果 |
| | B2 时间线一致性 | 时序冲突、时间流速 |
| | B3 上下文连贯性 | 前后呼应、伏笔回收 |
| | B4 叙事视角一致性 | POV越界、内心读取 |
| **C类 阅读体验** (×0.25) | C1 节奏与爽点 | 节奏曲线、期待感、开篇吸引力 |
| | C2 人物塑造 | 立体度、成长弧、差异化 |
| | C3 世界观自洽 | 设定矛盾、体系一致性 |
| | C4 对话质量 | 同质化、功能化、弦外之音 |
