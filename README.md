# novel-audit-skill

网文审计技能 - 系统性审计网络小说文本

## 功能特性

- **11个审计维度**：文字质量、AI痕迹、重复雷同、情节合理性、时间线一致性、上下文连贯性、叙事视角一致性、节奏与爽点、人物塑造、世界观自洽、对话质量
- **双模式**：单章详细审计 + 多章跨章对比审计
- **去AI化改写建议**：每条问题附带 ❌原文 → ✅建议 → 💡思路
- **加权评分**：A类×0.35 + B类×0.40 + C类×0.25，输出综合评级
- **题材覆盖**：玄幻/都市/悬疑/言情/历史等全题材通用

## 安装

将本技能安装到 WorkBuddy/Cursor 等支持 SKILL.md 的 AI 编程助手即可使用。

## 评分标准

| 综合分 | 评级 | 建议 |
|--------|------|------|
| 8.5-10 | 优质 | 可直接发布 |
| 7.0-8.4 | 良好 | 处理高优问题后发布 |
| 5.5-6.9 | 及格 | 处理高优+中优问题后发布 |
| 4.0-5.4 | 待改进 | 建议暂缓发布 |
| 0-3.9 | 需大改 | 建议重写 |
