# AI写作特征模式库

本文档收录网络小说中常见的AI生成语言模式，供审计时识别和判断使用。

**使用说明：** 单独出现某一类特征不能判定为AI生成；当同一文本中3个以上类别的特征同时密集出现，才判定为"AI痕迹明显"。

---

## 一、高频套话词库

以下词汇/短语在AI生成文本中出现频率异常高，正常人类写作中使用频次远低于此。

### 1.1 情绪表达套话（AI最爱用的情绪描写）

```
心中涌现出一股...       （情绪量化套话）
眼眸深处闪烁着...       （眼睛描写公式）
嘴角微微上扬            （微笑描写高频词）
眸子微动/眼眸微闪       （眼睛反应套话）
深吸一口气，平复心情    （压制情绪套话）
心头一紧                （紧张反应套话）
心中一动                （被触动套话）
五味杂陈                （复杂情绪套话）
百感交集                （情绪总结套话）
莫名的感动/感慨          （无来由的情绪套话）
热泪盈眶/眼眶湿润       （感动描写套话）
```

### 1.2 思维/意识套话（AI喜欢的"主角内心OS"模式）

```
他深知...               （旁白式内心总结）
她明白...               （旁白式内心总结）
他清楚地知道...         （旁白式内心总结）
这一刻，他终于明白...   （顿悟套话）
他心里清楚，...         （强调式内心总结）
不得不承认...           （内心博弈套话）
他暗自下定决心...       （立志套话）
这一切都是命运的安排    （命运总结套话）
命运的齿轮开始转动      （命运隐喻套话）
```

### 1.3 行动/反应套话

```
不动声色               （掩藏情绪高频词）
淡淡地说               （平静表达套话，极高频）
轻描淡写               （轻松应对套话）
云淡风轻               （从容态度套话）
冷冷地扫了一眼         （审视套话）
缓缓开口               （说话前置套话，极高频）
沉默片刻               （思考前置套话）
沉声道                 （严肃说话套话）
淡然道/淡漠道          （冷淡说话套话，极高频）
嗤之以鼻               （鄙视套话）
```

### 1.4 场景/环境描写套话

```
阳光透过窗户洒落...     （晨光开篇套话）
空气中弥漫着...的气息   （气氛描写套话）
夜风轻轻吹过...         （夜晚氛围套话）
沉默像潮水一样蔓延...   （沉默比喻套话）
时间仿佛凝固了          （紧张时刻套话）
空气都变得凝重起来      （紧张氛围套话）
寂静笼罩了整个...       （安静描写套话）
```

### 1.5 结构/过渡套话

```
正当此时                （情节转折套话，极高频）
就在这时                （情节转折套话，极高频）
就在这千钧一发之际      （紧张转折套话）
与此同时                （并行叙事套话）
然而，谁也没想到...     （转折套话）
这一切，都只是...刚刚开始   （章末收尾套话）
```

### 1.6 旁观者惊叹模板

```
周围的人都惊呆了        
在场所有人倒吸一口冷气  
没有人想到/谁也没想到   
人群中爆发出一阵骚动    
四周陷入一片寂静        
所有人的目光都聚焦在... 
议论纷纷                
面面相觑                
```

## 二、AI句式模板库

### 2.1 "三段式描写"模板

AI极爱用"描写X→描写Y→升华情绪"的三段式结构：

```
[视觉描写]，[听觉/嗅觉描写]，[主观感受总结]。
```

❌ 典型AI三段式：
```
阳光透过树叶的缝隙洒落，斑驳的光影在地面跳舞，空气中弥漫着泥土的气息，
让人心旷神怡，仿佛所有的烦恼都随风而去。
```

**识别特征**：环境描写→气味描写→情绪总结，句式工整，收尾必有"情绪升华"。

---

### 2.2 "排比三连"模板

AI写作常出现整齐的三连排比，且三个分句长度几乎相同：

```
有人说，人生是一场旅途；有人说，人生是一本书；有人说，人生是一幕戏。
```

```
他不后悔选择了这条路，不后悔付出了这些代价，不后悔成为了现在的自己。
```

**识别特征**：三句结构完全平行，长度基本相同，语义相近但表达不同，有刻意感。

---

### 2.3 "转折+感悟"模板

AI喜欢在关键情节后安排一段"主角感悟"，格式固定：

```
[事件发生]。
[主角沉默片刻]。
[主角内心OS：这件事让他深刻意识到...]。
[主角下定决心/改变想法]。
```

❌ 典型例子：
```
战斗结束了。
李明站在废墟中，沉默良久。
这一刻，他终于明白，真正的强大不是击败对手，而是超越自己。
他握紧了手中的剑，眼神前所未有地坚定。
```

**识别特征**：事件→沉默→感悟→决心，四步公式，感悟部分有强烈的"书本金句"感。

---

### 2.4 "高潮衬托"模板

AI惯用旁观者的反应来衬托主角的强大，而非直接描写主角：

```
[主角做了某事]。
[旁观者群体反应：惊呆/震惊/倒吸冷气]。
[旁观者代表开口表达震惊："这...这怎么可能！"]。
[再强调主角的淡然/平静]。
```

❌ 典型例子：
```
那一剑，斩断了山峰。
台下的弟子们全都惊呆了，没有人说话。
"这...这怎么可能..."有人喃喃自语，声音颤抖。
而陈明只是平静地收起了手中的剑，转身离去。
```

**识别特征**：主角行为→群体震惊→代表性发言（带"..."和"怎么可能"）→主角的"淡然"对比。

---

### 2.5 "深情凝视"模板

AI描写人物对视/凝视的固定公式：

```
[人物A]抬起头，与[人物B]的目光相对。
[视线/眸子的描写]。
[内心感受：莫名/奇异的...]。
```

❌ 典型例子：
```
她抬起头，恰好对上了他深邃的目光。
那双眼睛里，似乎藏着什么，让她心头无端一跳。
她不知道这是什么感觉，但她知道，她从未有过这种感觉。
```

---

### 2.6 "章末收尾"模板

AI的章末极度依赖两种收尾方式：

**A型（悬念型）：**
```
[事情发生了变化]。
这一切，都只是刚刚开始……
```

**B型（抒情型）：**
```
[主角做了一个动作]，望向[远处/天空]。
他的眼神深邃，心中百感交集。
```

**识别特征**：省略号+泛情绪词（"百感交集"、"思绪万千"、"种种情感"）收尾。

## 三、AI写作结构特征

### 3.1 段落长度高度均匀

人类写作中段落长度参差不齐，有一行的短段，也有十行的长段。AI生成文本段落长度往往非常均匀（每段3-5行），看起来格式整齐但单调。

### 3.2 每段必有情绪收尾

AI倾向于在每段结尾安排一个情绪/感受句，造成每段都在"总结"，缺乏留白。

### 3.3 过渡依赖"转折词"

AI频繁使用过渡词来连接段落：
```
然而 / 不过 / 但是 / 与此同时 / 这时 / 正当此时 / 就在这时
```
如果一章文本中这类词出现频率极高，是AI写作的重要特征。

### 3.4 冲突解决过于顺滑

AI不善于处理真正的困境：
- 主角遇到的困难总在合适的时候化解
- 对手犯错的时机总是恰到好处
- 帮助总在需要的时候出现

### 3.5 人物对话缺乏个性

AI的对话有强烈的同质化特征：
- 所有角色的说话方式相似，难以通过对话判断是谁在说
- 对话内容信息密度高，但缺乏"废话"（人类真实对话充满废话）
- 情感表达直白，没有迂回或暗示

## 四、去AI化改写策略

### 策略1：用具体细节替换抽象情绪

| AI版本 | 去AI化改写方向 |
|---|---|
| "心中涌现出一股感动" | 写一个微小的生理反应（手抖了一下/鼻子酸了） |
| "眼眶湿润" | 写他用了什么动作掩盖（低下头/转过身/假装揉眼睛） |
| "心头一紧" | 写他做了什么无意识的动作（攥紧了拳头/停顿了一秒） |

### 策略2：打散工整排比

| AI版本 | 去AI化改写方向 |
|---|---|
| 三句完全平行的排比 | 改成长短错落，第三句用短句或反问收尾 |
| "有人说A，有人说B，有人说C" | 改成：听说有人觉得A，也听说有人说是B，至于C——他没想过 |

### 策略3：删掉旁观者惊叹

| AI版本 | 去AI化改写方向 |
|---|---|
| "在场所有人都惊呆了" | 删掉，改为一两个具体人物的具体反应 |
| "人群中爆发出骚动" | 改成：某个人的手抖了一下，茶杯碎在地上 |

### 策略4：替换章末套话

| AI版本 | 去AI化改写方向 |
|---|---|
| "这一切，都只是刚刚开始……" | 改成一个具体的、出乎意料的细节 |
| "他望向远处，心中百感交集" | 改成：他往衣兜里摸了摸，掏出一枚硬币，反复在指尖转动 |

### 策略5：让内心OS更像人在想

| AI版本 | 去AI化改写方向 |
|---|---|
| "他终于明白，真正的强大是…" | 改成碎片化的真实思维："他想了好半天，也想不出什么大道理。就是觉得——累了。" |
| "命运的齿轮开始转动" | 删掉，直接写下一个行动 |

## 五、参考：正常人类写作 vs AI写作对比

### 例一：危险结束后的反应

**AI版：**
```
战斗结束了。李明站在废墟中，沉默良久。
这一刻，他终于明白，真正的强大不是击败对手，而是超越自己。
他握紧了手中的剑，眼神前所未有地坚定。
```

**人类写法（参考）：**
```
最后一个倒下去的时候，李明还没反应过来。
他低头看了看手，不知道从哪里蹭来的血，已经快干了，颜色很深。
周围很安静。他想起来应该检查伤势，但脚下没动。
```

---

### 例二：角色对视

**AI版：**
```
她抬起头，恰好对上了他深邃的目光。
那双眼睛里，似乎藏着什么，让她心头无端一跳。
她不知道这是什么感觉，但她知道，她从未有过这种感觉。
```

**人类写法（参考）：**
```
她抬头，撞进他的视线。
一秒。两秒。
她先移开了眼，假装低头找什么东西。
```

## 六、补充AI模式库（2026年5月更新）

以下模式来自对 qu-ai-wei 项目、AI文本检测研究、网文编辑审稿实践的综合参考，补充前五节未覆盖的AI写作特征。

### 6.1 华丽意象词堆叠

AI在描写场景时极爱用"辞藻华丽但空洞"的意象词，营造虚假的文学感。

```
璀璨      熠熠生辉    画卷       华章       乘风破浪
扬帆起航  落英缤纷    繁星似锦    云蒸霞蔚    波光粼粼
气势磅礴  震撼人心    惊天动地    气吞山河    如梦似幻
如诗如画  美不胜收    令人窒息    肃然起敬    悠悠岁月
```

**识别特征**：一段描写中堆叠 ≥3 个此类词，且无具体细节支撑。

❌ 典型AI：
```
璀璨的星空下，波光粼粼的湖面如梦似幻，令人窒息的美景让他久久不能忘怀。
```

✅ 人味：
```
他坐在湖边，仰头看了很久。
星星倒映在水里，一动不动，像被冻住了。
```

---

### 6.2 "的的不休"

AI生成的句子中"的"字使用密度异常高，尤其是定语从句堆叠。

**判定标准**：单个句子内出现 ≥4 个"的"字。

❌ `那个穿着红色衣服的看起来很生气的女孩的朋友`（5个"的"）
✅ `穿红衣服那女孩的朋友，一脸不高兴`

**更严重的——"的"字从句堆叠**（翻译腔残留）：
❌ `他是一个被所有人认为是不可能成功的年轻人的唯一的朋友`
✅ `所有人都觉得那年轻人没戏。偏偏他是唯一没放弃的朋友。`

---

### 6.3 抽象万能动词泛滥

AI在叙事中大量使用"放到任何话题都能套"的抽象动词，而非具体的动作描写。

```
进行（进行研究/进行讨论）  开展（开展工作/开展调查）
实施（实施部署/实施计划）  发挥（发挥作用/发挥优势）
实现（实现突破/实现目标）  推动（推动发展/推动变革）
达成（达成共识/达成目标）  提升（提升效率/提升品质）
打造（打造团队/打造品牌）  构建（构建体系/构建框架）
赋能（赋能产业/赋能团队）  激发（激发潜力/激发活力）
```

**判定标准**：一段文字中抽象万能动词占所有动词的 1/3 以上。

**网文场景中的表现**：
❌ `他进行了深入的修炼，实现了突破，发挥了自己的潜力。`
✅ `他闭关三天，灵气走完第一个大周天的时候，浑身骨头都在响。`

---

### 6.4 翻译腔残留

AI生成中文小说时，常残留英文语法结构的痕迹。

**常见表现：**

| 翻译腔特征 | 示例 | 修正 |
|---|---|---|
| 过度使用代词"他/她" | "他拿起了他的剑，他看向她" | 删掉多余代词 |
| "作为一个X..."开场 | "作为一个修士，他知道这很危险" | "他修了十年道，这种危险一眼就看出来了" |
| 被动句密度过高 | "他被带到了大厅，被告知了结果" | "有人把他带到大厅，告诉了他结果" |
| 介词短语开头过长 | "与那件已经过去很久的事情相关的一切" | "那件事过去很久了，但所有相关的一切……" |
| 英文成语直译 | "在最后一秒"（last second） | "最后关头 / 千钧一发之际" |

---

### 6.5 否定对举句

AI极爱用"不是X，而是Y"的工整对称结构来做定义或总结。

```
这不是结束，而是新的开始。
他不是在逃避，而是在等待。
她想要的不是同情，而是尊重。
这不仅仅是运气，更是实力的体现。
```

**识别特征**：短段落内出现 ≥2 处"不是X，而是Y"结构，或全文出现 ≥4 处。
注意：偶尔使用是正常的，密集出现才是AI特征。

---

### 6.6 内容通用、细节缺失

AI训练数据覆盖广但深度不足，生成内容往往"什么都提到了一点，但什么都不具体"。

**表现：**
- 描写城市："繁华的街道两旁是各种各样的商铺"（哪些商铺？卖什么？）
- 描写美食："桌上摆满了美味佳肴"（什么菜？什么味道？）
- 描写武功："他使出了一招精妙的剑法"（什么招式？怎么精妙？）

**判定标准**：连续 200 字以上的描写中无任何一个具名细节（具体名称/具体数量/具体颜色/具体声音）。

---

### 6.7 节奏均质化（句长无变化）

人类写作自然会有句子长短的变化——有短促有力的，也有悠长的描写。AI文本句子长度高度均匀。

**判定标准**：连续 5 句以上字数在 15-30 字之间波动不超过 5 字。

**更深层表现**：每个段落的长度也高度一致（都是 3-5 行），无明显的长段/短段交替。

---

### 6.8 AI"过度干净"特征

这是最隐蔽也最常被忽略的AI特征：**文本太过完美，毫无瑕疵**。

**表现：**
- 全文无任何口语化表达、方言、俚语
- 无任何"不完美"的人类表达（犹豫、自嘲、跑题、废话）
- 无任何个人化的"毛边"（只有作者本人会用的独特表述）
- 情绪表达永远得体、适度，从不失态

**判定标准**：在 3000+ 字的文本中，找不到任何一句"AI不敢写的话"（如自嘲、承认不确定、私人细节、不合时宜的观点）。

**这不是AI痕迹的直接证据，但当与其他AI特征同时出现时，可信度极高。**

---

### 6.9 情感抽离/过于平衡

AI在处理争议或冲突时，习惯性地"端水"——让双方都有理。

**网文场景表现：**
- 冲突场景中，给反派也安排合理的动机（没有纯粹的恶人）
- 每次重大选择后，主角都要反思"也许对方也没错"
- 争执双方都有道理，读者无法判断作者更偏向谁

**判定标准**：在需要明确立场的关键情节中，文本表现出过度的平衡和犹豫。

**注意**：有些作品风格本身就是"灰色"叙事，需区分"刻意为之"和"AI默认行为"。如果全文所有冲突都是"双方有理"的模式，则更可能是AI特征。

## 七、AI模式快速检测清单

审计时，可按以下清单快速扫描文本中是否存在AI特征（勾选命中项，≥5项则AI痕迹明显）：

**词汇层面：**
- [ ] 情绪套话（心中涌现/眼眸闪烁/嘴角微微上扬）
- [ ] 华丽意象词堆叠（璀璨/熠熠生辉/如梦似幻）
- [ ] AI高频行动词（缓缓开口/淡淡地说/不动声色/沉默片刻）
- [ ] 抽象万能动词（进行/开展/实现/发挥/推动/打造）
- [ ] 转折/过渡词过密（正当此时/就在这时/然而/与此同时）

**句式层面：**
- [ ] 三段式描写（视觉→听觉→情绪升华）
- [ ] 排比三连（三句结构完全平行）
- [ ] 转折+感悟模板（事件→沉默→内心OS→决心）
- [ ] 否定对举句（"不是X，而是Y"≥2处）
- [ ] "的的不休"（单句≥4个"的"字）

**结构层面：**
- [ ] 段落长度高度均匀（每段3-5行）
- [ ] 句长节奏均质化（连续5句15-30字）
- [ ] 每段末尾必有情绪收尾句
- [ ] 高潮衬托模板（主角行为→群体震惊→代表发言→主角淡然）
- [ ] 章末省略号+泛情绪收尾

**内容层面：**
- [ ] 旁观者惊叹模板（在场所有人惊呆/倒吸冷气/面面相觑）
- [ ] 内容通用无细节（连续200字无具名细节）
- [ ] 文本过度干净（无口语/无方言/无毛边/无自嘲）
- [ ] 情感抽离（所有冲突都"双方有理"）
- [ ] 翻译腔残留（代词滥用/被动句过多/英文成语直译）
