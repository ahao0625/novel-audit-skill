# 评分标准档位说明

本文档定义 novel-audit 技能各维度0-10分的评分标准，确保评分有据可依、尺度统一。

---

## 评分总体原则

- **评分基于证据**：每个评分必须在报告中有对应的具体问题或优点作为依据
- **宁低勿高**：有明显问题时，宁可给低分，不要为了"给人信心"而虚高
- **区分严重程度**：影响阅读体验的问题（如情节BUG、严重AI化）比文字小问题权重更高
- **不完美≠低分**：6分以上都属于合格范围，不代表失败

---

## 分值档位定义

| 分值 | 等级 | 含义 |
|------|------|------|
| 9-10 | 优秀 | 专业水准，几乎无可挑剔 |
| 7-8  | 良好 | 质量不错，有少量可改进项 |
| 5-6  | 及格 | 基本可读，但有明显问题需处理 |
| 3-4  | 需改进 | 问题较多，影响阅读体验 |
| 1-2  | 较差 | 问题严重，建议大幅修改 |

---

## 各维度评分细则

### A1 文字基础质量

| 分值 | 判定标准 |
|------|---------|
| 9-10 | 语言精准流畅，无病句、无错别字，描写有画面感，节奏感好 |
| 7-8  | 整体通顺，有2-4处小问题（轻微冗余/偶有套话），但不影响阅读 |
| 5-6  | 有明显流水账段落，或5处以上语言问题，但大部分内容可读 |
| 3-4  | 大量流水账/病句/形容词堆砌，阅读体验受损 |
| 1-2  | 语言混乱，大量错误，难以正常阅读 |

**特殊加分项**：有精彩的比喻、有画面感的细节描写、节奏处理出色 → 可在基准分上+1

---

### A2 重复雷同检测

| 分值 | 判定标准 |
|------|---------|
| 9-10 | 词汇丰富，无明显重复，每个场景结构各有特色 |
| 7-8  | 有1-2个高频词使用偏多，或有1处结构轻微雷同 |
| 5-6  | 有3-5处明显重复（词汇/句式/情节结构），影响阅读新鲜感 |
| 3-4  | 重复问题突出，多个场景高度雷同，读者会有"看过了"的感觉 |
| 1-2  | 大规模复制粘贴式重复，严重影响阅读体验 |

---

### A3 AI痕迹识别

| 分值 | 判定标准 |
|------|---------|
| 9-10 | 语言有个人风格，无明显AI特征词/句式，叙事自然流畅 |
| 7-8  | 有1-2处AI特征词/句式，但整体文风较有个性 |
| 5-6  | 有3-5处AI特征，部分段落套路感明显，但还有人味 |
| 3-4  | AI特征密集（6-9处），文章整体套路感强，像是套模板写出来的 |
| 1-2  | 高度AI化，大量套话/模板/旁观者惊叹，文风极度公式化 |

---

### B1 情节合理性

| 分值 | 判定标准 |
|------|---------|
| 9-10 | 情节发展自然，人物行为动机充分，无明显逻辑BUG |
| 7-8  | 整体合理，有1处小的动机不足或轻微的上帝之手 |
| 5-6  | 有2-3处明显的情节BUG或动机缺失，需要补充铺垫 |
| 3-4  | 情节问题较多（4-5处），多处上帝之手或因果跳跃 |
| 1-2  | 情节混乱，逻辑严重不自洽，读者完全无法信服 |

---

### B2 时间线一致性

| 分值 | 判定标准 |
|------|---------|
| 9-10 | 时间信息精确一致，无矛盾，时间感清晰 |
| 7-8  | 时间线整体清晰，有1处小的不一致（轻微矛盾或不够精确） |
| 5-6  | 有2-3处时间/季节/年龄描述矛盾，需要修正 |
| 3-4  | 时间线混乱，多处矛盾，读者难以建立清晰的时间感 |
| 1-2  | 时间线严重混乱，前后矛盾多且明显 |

**注意**：如果文本本身不涉及明确时间信息，该维度默认8分（无法检测则不扣分）

---

### B3 上下文连贯性

| 分值 | 判定标准 |
|------|---------|
| 9-10 | 设定/情绪/人物状态前后完全一致，衔接自然流畅 |
| 7-8  | 整体连贯，有1处轻微的设定遗忘或情绪承接不够细腻 |
| 5-6  | 有2-3处连贯性问题（设定遗忘/情绪突变/状态矛盾） |
| 3-4  | 连贯性较差，读者频繁感到"前面不是说过..."的困惑 |
| 1-2  | 严重不连贯，与前文设定大量矛盾，像是另起炉灶 |
|  |  |  |

---

### B4 叙事视角一致性

| 分值 | 判定标准 |
|------|---------|
| 9-10 | 视角全程一致，无限知越界，视角切换有明确分隔 |
| 7-8  | 整体一致，有1处轻微视角越界（不影响理解） |
| 5-6  | 有2-3处视角问题，读者可能感到困惑 |
| 3-4  | 视角混乱，多次越界或无预告切换 |
| 1-2  | 视角完全失控，读者无法判断谁在感知 |

---

### C1 节奏与爽点设计

| 分值 | 判定标准 |
|------|---------|
| 9-10 | 节奏张弛有度，爽点设计精准，章末有强力钩子，情绪曲线完整 |
| 7-8  | 节奏整体不错，偶有拖沓，有明确爽点但力度可加强 |
| 5-6  | 节奏有明显问题（拖沓段落 or 缺乏爽点），章末钩子一般 |
| 3-4  | 节奏单一，大量拖沓，读者难以坚持读完，章末无吸引力 |
| 1-2  | 节奏极差，几乎无情节推进，没有任何爽点或悬念 |

---

### C2 人物塑造一致性

| 分值 | 判定标准 |
|------|---------|
| 9-10 | 人物立体真实，性格一致且有层次，对话有个性辨识度 |
| 7-8  | 人物整体一致，有1-2处口吻稍有偏差，但不影响整体 |
| 5-6  | 有2-3处性格/口吻不一致，或1处较明显的性格矛盾 |
| 3-4  | 人物塑造问题明显，性格前后矛盾，对话千人一面 |
| 1-2  | 人物完全脸谱化/无个性，或性格设定严重不自洽 |

---

### C3 世界观自洽性

| 分值 | 判定标准 |
|------|---------|
| 9-10 | 世界规则体系严密自洽，力量体系/社会逻辑前后完全一致 |
| 7-8  | 整体自洽，有1处规则细节不够严密但不影响读者理解 |
| 5-6  | 有2-3处规则矛盾或设定遗忘，影响沉浸感 |
| 3-4  | 规则体系混乱，多处矛盾，读者难以建立对世界的信任 |
| 1-2  | 世界观严重不自洽，规则随情节需要随意修改 |

**注意**：如果文本不涉及特定世界观（如写实现代文），该维度默认7分

---

### C4 对话质量专项

| 分值 | 判定标准 |
|------|---------|
| 9-10 | 对话有辨识度，信息密度合理，弦外之音丰富 |
| 7-8  | 整体不错，有1-2处对话同质化或信息密度过高 |
| 5-6  | 有3-4处对话问题，角色声音区分度不足 |
| 3-4  | 对话问题明显，所有人说话方式雷同，功能化严重 |
| 1-2  | 对话完全无辨识度，或信息过载严重影响阅读 |

---

## 综合评分加权规则

```
综合评分 = A类均分×0.35 + B类均分×0.40 + C类均分×0.25

A类均分 = (A1 + A2 + A3) / 3
B类均分 = (B1 + B2 + B3 + B4) / 4
C类均分 = (C1 + C2 + C3 + C4) / 4
```

**权重说明：**
- B类（叙事逻辑）权重最高（0.40）：因为情节BUG和时间线矛盾是最影响阅读体验的核心问题
- A类（文本质量）权重次之（0.35）：文字质量直接影响阅读感受
- C类（读者体验）权重最低（0.25）：节奏/人物问题相对可接受，且更主观

**总体评级参考：**

| 综合分 | 总体评级 | 发布建议 |
|--------|---------|---------|
| 8.5-10 | 优质 | 可直接发布，锦上添花式修改 |
| 7.0-8.4 | 良好 | 处理高优问题后可发布 |
| 5.5-6.9 | 及格 | 处理高优和中优问题后发布 |
| 4.0-5.4 | 待改进 | 需要较多修改，建议暂缓发布 |
| 0-3.9 | 需大改 | 建议重写或大幅修改，不建议发布 |
